BMI with objects

Kasun and Chamal are trying to compare their BMI(Body Mass Index), which is calcualted using the formula; BMI = mass/height^2; (Mass is kg and height is meter)

  1. For each of them create an object with full name, mass and height
  2. Then, add a method to object to calculate the BMI. Save the BMI to the object and also return it from the method
  3. In the end, display the highest BMI with the full name. Don't forget they have might the same BMI.
